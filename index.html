Sobre

Este projeto é um clone do Lubuntu 20.04 LTS com temas e aparência do sistema Windows®. É para usuários que querem migrar para um SO gratuito e open-source, mas não tem habilidades técnicas. Comece com o WinuniX, depois migre de vez para a liberdade!
